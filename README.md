\######CALCULADORA INTERACTIVA JS######
Una aplicación web sencilla y funcional para realizar operaciones matemáticas básicas con gestión de historial en tiempo real.

# DESCRIPCIÓN

Este proyecto es una herramienta de cálculo que utiliza JavaScript para procesar datos ingresados por el usuario. La interfaz es minimalista y prioriza la visualización limpia de los resultados directamente en el documento HTML.

## FUNCIONES PRINCIPALES

Cálculo Dinámico: Procesa sumas, restas, multiplicaciones y divisiones mediante ventanas emergentes.

Gestión de Historial: Almacena cada operación en un arreglo interno con un ID único.

Validación de Seguridad: Controla el ingreso de datos no numéricos y bloquea divisiones por cero.

Interfaz Inteligente: La pantalla se actualiza automáticamente para mostrar solo la información relevante del momento.

### INSTRUCCIONES DE USO

Iniciar: Haz clic en el botón "Realizar Cálculo".

Entrada: Ingresa los valores solicitados en las ventanas del navegador.

Resultado: El resultado aparecerá en la parte central de la página de forma inmediata.

Consulta: Presiona "Ver Historial" para desplegar la lista completa de tus cálculos anteriores.

Limpiar: Usa el botón de limpieza para vaciar la información visual del HTML.

#### ESPECIFICACIONES TÉCNICAS

Lógica: Basada en un objeto de operaciones y funciones flecha.

Manipulación DOM: Uso de innerHTML para actualizar el contenido sin recargar la página.

Almacenamiento: Uso de arreglos y objetos para la persistencia de datos durante la sesión.
